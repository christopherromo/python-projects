## Reflection 💭

**The Process:** The order these projects were created is the same order of the classes as listed above. "Python Programming" focused on Python fundamentals, exploring concepts like variables, control structures, functions, classes, objects, and libraries. For "Discrete Structures", explored concepts include set theory, propositional logic, exponential growth, bases, and graph representation. For "Computational Linear Algebra", explored concepts include matrix math, transformations, and several model representations. For "Design & Analysis of Algorithms", different sorting algorithms were explored. For "Computer Architecture I", branch prediction was explored. For "Computability, Automata, Formal Languages", deterministic finite automata were explored. Each project was usually completed in a 1-2 week period, though there are exceptions.

**Key Takeaways:** Only "Python Programming" specifically focused on the language itself, while the rest of the classes had their own topics. Python is a multi-paradigm language, meaning many concepts can be applied to and from other languages. I started by learning how to setup my Python environment and write Python code. Since Python is a simpler language, I was able to jump in quicker and use this language for several courses. With Python having a large range of libraries, exploring new topics was easy. Overall, I not only learned how to program with Python, I used it as tool to learn several other concepts in computer science.

## Running the Projects 🎬

1. Clone the repository.

2. Ensure Python is installed on your computer.

3. Open the `python-projects` directory in Visual Studio Code.
    - Ensure the "Python Extension Pack" is installed.

4. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

5. Navigate to the desired Python project.

6. Click the run button in the editor to run the file.
